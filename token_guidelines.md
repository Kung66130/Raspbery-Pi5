# AI Interaction Guidelines (Token Optimization)

## Model Commitment
- **Response Style**: Concise, essential information only.
- **Focus**: Technical details and direct task execution.
- **Tone**: Minimalist and precise.

## User Best Practices for Saving Tokens
1. **Clear Instructions**: Specify exact files and lines (e.g., "Change line 10 in config.py to X").
2. **Task Isolation**: Focus on one specific objective per turn.
3. **Context Management**: Start new conversation threads for unrelated tasks.
4. **Targeted Tools**: Prefer incremental edit tools over full file rewrites.
