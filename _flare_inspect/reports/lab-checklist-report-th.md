# รายงานเช็กลิสต์งานทดลอง

> เอกสารนี้จัดรูปแบบจากรายการเช็กลิสต์ที่ผู้ใช้ส่งมา เพื่อให้ง่ายต่อการติดตามสถานะงาน

## 1) Preparation
- [ ] เตรียมอุณหภูมิ 80°C
- [ ] เติม tween
- [ ] จดรายละเอียดการละลาย
- [ ] ถ่ายรูปนิ่ง
- [ ] ถ่ายวิดีโอ
- [ ] จุ่มแท่งแก้ว
- [ ] จุ่มกระจกสไลด์
- [ ] เคลือบเม็ดปุ๋ย 11 เม็ด
- [ ] เคลือบกระจก 3 แผ่น

## 2) Viscosity Test
- [ ] 80°C #1
- [ ] 80°C #3
- [ ] 70°C #1
- [ ] 70°C #3
- [ ] อัปวิดีโอ
- [ ] ลงผล

## 3) Adhesion Test
- [ ] ถ่ายภาพก่อนแปะ
- [ ] แปะเทป
- [ ] อัปรูป
- [ ] ลงผล

## 4) Hardness Test
- [ ] กดน้ำหนัก
- [ ] อัปรูป
- [ ] ลงผล

## 5) Hydrophobicity Test
- [ ] ถ่ายหยดน้ำ
- [ ] อัปวิดีโอ

## 6) Humidity Test
- [ ] เตรียมกล่อง
- [ ] ถ่ายภาพ
- [ ] ชั่งน้ำหนัก
- [ ] ลงรูป

## 7) Dye Penetration
- [ ] ถ่ายภาพ
- [ ] จุ่มสี
- [ ] ถ่ายภาพหลังจุ่ม
- [ ] ผ่าเม็ด
- [ ] ถ่ายภาพหลังผ่า
- [ ] อัปรูป
- [ ] ลงผล

## 8) Open Air Test
- [ ] ถ่ายภาพ T0
- [ ] ถ่ายภาพ T end
- [ ] อัปรูป
- [ ] ลงผล

## 9) Glass Slide
- [ ] ถ่ายภาพก่อนเคลือบสี
- [ ] ถ่ายภาพหลังเคลือบสี

## 10) Glass Rod
- [ ] ถ่ายภาพก่อนเคลือบสี
- [ ] ถ่ายภาพหลังเคลือบสี
- [ ] ถ่ายวิดีโอหลังเคลือบสี
