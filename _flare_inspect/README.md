# Flare Social (แอปโซเชียลสไตล์ Litmatch)

แอปพลิเคชันโซเชียลเน็ตเวิร์กสุดพรีเมียมที่ออกแบบมาเพื่อสร้างความสัมพันธ์และความรู้สึกที่แท้จริง

## ฟีเจอร์หลัก
- **Soul Game**: จับคู่กับคนแปลกหน้าเพื่อสนทนากันอย่างลึกซึ้งเป็นเวลา 5 นาที
- **Voice Game**: เชื่อมต่อผ่านเสียงเพื่อการโต้ตอบที่มีความหมาย (กำลังพัฒนา)
- **Party Rooms**: ห้องแชทเสียงกลุ่มพร้อมกิจกรรมที่หลากหลาย (กำลังพัฒนา)
- **Moments Feed**: แบ่งปันเรื่องราวในชีวิตประจำวันและความคิดของคุณ

## เทคโนโลยีที่ใช้
- **Frontend**: React (Vite)
- **Styling**: Vanilla CSS พร้อมดีไซน์ Glassmorphism
- **Icons**: Lucide React
- **Animations**: Framer Motion

## เริ่มต้นใช้งาน
1. `npm install`
2. `npm run dev`

## โครงสร้างโปรเจกต์
- `src/App.jsx`: โครงสร้างหลักของแอปพลิเคชันและการจัดการเส้นทาง (Routing)
- `src/index.css`: ระบบการออกแบบ (Design System) และสไตล์ส่วนกลาง
- `src/components/`: ส่วนประกอบ UI แบบโมดูลาร์ (SoulGame, PartyRoom, ฯลฯ) - เร็วๆ นี้
